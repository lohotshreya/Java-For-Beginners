# Java-For-Beginners

This repository contains basic Java programs designed to build a strong foundation in object-oriented programming and problem-solving.

## Goals
- Learn Java from scratch
- Understand OOP principles
- Build logical thinking and clean code habits
- Prepare for technical interviews and internships

## Repository Structure
- basics/ → syntax, variables, loops
- oops/ → classes, objects, inheritance, polymorphism
- collections/ → ArrayList, HashMap, Set
- mini_exercises/ → small practice problems

## Learning Approach
- Beginner-friendly examples
- Focus on clarity and structure
- Regular practice and incremental learning
